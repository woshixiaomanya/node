项目名称：基于 Node.js 的轻量化离线 RAG 智能问答系统
项目简介：独立搭建 Node.js 版 RAG（检索增强生成）架构，无需依赖在线大模型、向量库及 GPU，纯本地离线运行，专注解决前端转AI场景的知识库智能查询需求，实现“文档加载→文本拆分→语义检索→对话交互”的完整闭环。
核心技术与实现：
- 文档处理：基于 Node.js fs 模块实现本地长文档读取，自定义分句拆分算法，按句子分组切片，避免长文本语义断裂，提升检索精准度。
- 语义检索：自研离线语义关键词映射词典，实现同义句（如“实战”→“项目实操”）智能匹配，解决传统关键词检索死板、漏检的痛点。
- 交互设计：基于 readline 模块实现终端交互式对话，支持多轮对话记忆存储，保留历史聊天上下文，提升用户交互体验。
- 工程化适配：零第三方依赖，无需联网下载模型，可直接通过 node 命令运行，支持打包为桌面可执行程序，适配无 Node.js 环境的设备。
项目价值：掌握 Node.js 生态下 RAG 技术的核心实现逻辑，实现轻量化、高可用的离线问答系统，可快速迁移到企业内部文档、培训手册等智能查询场景，契合前端/全栈转AI的技术发展方向。
